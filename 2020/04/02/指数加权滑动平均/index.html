我们假设 $\beta = 0.9$，按照降序排列，列出相应的等式了解一下它是如何计算每日气温的：
$
v_{100} = 0.9v_{99} + 0.1\theta_{100}\\
v_{99} = 0.9v_{98} + 0.1\theta_{99}\\
v_{98} = 0.9v_{97} + 0.1\theta_{98}\\
…
$

这就是我们对第 100 天数据的预测。我们可以看到，这个预测就是指数衰减函数乘以一个固定的数据$\theta$然后求和。

然后我们可能好奇，这到底是前几天的气温平均值呢？按照上面的例子来说，$0.9^{10}$ 大约等于 $\frac{1}{e}$，即经过10天的时间，指数衰减函数能降到峰值的 $\frac{1}{e}$ 左右，即在10天前的数据的权重不到当天数据权重的 $\frac{1}{e}$ 。因此这应该是前10 天气温的平均值，即在这个数据条件下，指数加权平均值只关注最近十天的气温值。

再举个例子，当 $\beta = 0.98$ 时，$0.98$ 的几次方才能使这个数据足够小呢，结果大概是 $0.98^{50}$ 约等于 $\frac{1}{e}$，即 50 天前数据的权重衰减为当前数据的 $\frac{1}{e}$，因此当 $\beta = 0.98$ 时，我们只关注前 50 个数据。

于是我们有 $ \frac{1}{1-\beta}$ 来衡量指数滑动平均关注哪些数据。
