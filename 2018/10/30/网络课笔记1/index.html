网络课笔记（1）

发表于 2018-10-30

主要介绍计算机网路的基本结构以及思科路由器的使用方法。

OSI七层网络协议

os层次 作用 协议 设备 数据
应用层 为用户提供应用端口 http、https、ftp、DNS message
表示层 为数据提供加密压缩服务 加密：AES、DES、3DES压缩：zip message
会话层 为数据处理提供平台 os：ios（internet operating system 思科的） message
传输层 为数据提供可靠和不可靠传输 TCP、UDP、SPX Fragment数据段
网络层 为数据提供逻辑寻址，提供转发差错控制和纠正 IP、IPX（Novell）、Appleta 路由器 Packet数据包
数据链路层 为数据提供物理寻址，提供交换功能，差错控制无法纠正 LAN：IEEE802.3、IEEE802.5、FDDI WAN：PPP、Frame-relay、HDLC 交换机（switch）、桥（bridge） Frame数据帧
物理层 为数据传输定义限速，线缆，电压 RS232、RJ45X.25 集线器Hub、中继器repeater、调制解调器modem 数据位

路由器的使用

连接真机设备

软件

secure CRT（WIN7及以上系统需要，XP自带超级终端）

RS232转接线驱动

硬件

console线，RS232-USB转接线、双绞线（直连和交叉各一根，直连用于交换机，交叉用于路由器）。

步骤

  1. 使用console线链接设备的“console”端口，RS-232连接电脑
  2. 安装RS-232转接线驱动，电脑查看端口。 “控制面板”--“设备管理器”--“端口（com和LPT）”
  3. 打开CRT，使用CRT桥接 “快速连接”--“协议：serial”--“端口com5”--“波特率9600”--“连接”
